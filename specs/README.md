# Specifications Directory

This directory contains the technical specifications for the Flynn system components.

## Spec Files

- `dev-environment.md`: Containerized development environment specification (Docker-based Linux build environment).
- `dev-machine.md`: Dev machine access instructions and system specs (Proxmox build server at 192.168.168.87).

## Planned Spec Files

- `architecture.md`: High-level system architecture and component interaction.
- `tuf-design.md`: Detailed design of the TUF implementation and security model.
- `api-specs.md`: Definitions for internal and external API endpoints.
- `data-models.md`: Schema definitions for persistent storage and configuration.
